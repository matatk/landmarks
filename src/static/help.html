How to change the keyboard shortcuts

  • Firefox: Menu ☰ → Add-ons → Extensions [on left-hand side, if not already activated] → Cog wheel [to the right of Manage Your Extensions] → Manage Extension Shortcuts.

  • Chrome: Menu ⋮ → More Tools → Extensions → Keyboard shortcuts [in the left-hand menu pane].

  • Opera: Opera Menu / Speed Dial → Extensions → Extension Keyboard Shortcuts.

  • Edge: Menu ⋯ → Extensions → Keyboard shortcuts [in the left-hand menu pane].

Features

Thanks for installing the Landmarks extension. With it, you can…

  • Quickly check how many landmark regions the page has, via the toolbar icon.
  • Explore landmarks via keyboard shortcuts.
  • Explore landmarks via the toolbar pop-up.
  • Explore landmarks via the browser sidebar (Firefox and Opera only).
  • Use the developer tools panel to inspect the code behind landmark regions.

You can change the following preferences.

  • How long the border is shown.
  • The border's colour.
  • The font size of the in-page landmark labels.
