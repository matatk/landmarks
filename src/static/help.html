Keyboard navigation ⚠warning

How to change the keyboard shortcuts

  • Firefox: Menu ☰ → Add-ons → Extensions [on left-hand side, if not already activated] → Cog wheel [to the right of Manage Your Extensions] → Manage Extension Shortcuts.

  • Chrome: Menu ⋮ → More Tools → Extensions → Keyboard shortcuts [in the left-hand menu pane].

  • Opera: Opera Menu / Speed Dial → Extensions → Extension Keyboard Shortcuts.

  • Edge: Menu ⋯ → Extensions → Keyboard shortcuts [in the left-hand menu pane].

Features

Thanks for installing the Landmarks extension. With it, you can…

  • Quickly check how many landmark regions the page has, via the toolbar icon.
  • Explore landmarks via keyboard shortcuts.
  • Explore landmarks via the toolbar pop-up.
  • Explore landmarks via the browser sidebar (Firefox and Opera only).
  • Use the developer tools panel to inspect the code behind landmark regions.

You can change the following preferences.

  • How long the border is shown.
  • The border's colour.
  • The font size of the in-page landmark labels.

What's new?

New in version …

  • When moving forward or backward through landmarks, navigation now starts from the element that has the focus (if there is one). You can still skip directly to the "main" region, and to any of the other landmarks via the pop-up and/or sidebar.
  • Add the first linting feature for developers: open the DevTools panel and it will warn you about duplicate unlabelled landmarks.
  • Move the DevTools panel to the DOM/Element Inspector tab, so that you can see the HTML and landmarks side-by-side.
  • Softer update notifications—when the extension is updated, instead of opening a new page, badge the browser toolbar button and include a note in the pop-up and/or sidebar.
  • Enhanced performance tests, so that the code can be kept as snappy as possible.
  • Edge users: version 2.8.0 was not available for Edge due to an issue with the system used for uploading extensions. The Edge team is looking into it, and I've developed a work-around, so we are back on now. Please check out what was new in version 2.8.0 so that you don't miss out on any of the info.

For full details, consult the change log.
